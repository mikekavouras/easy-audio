easy-audio
==========

Dead simple way to add sound effects to any website.

```javascript
  $('#button').easyAudio({
    sound: 'path_to_audio'
  });

  $('#button').easyAudio({
    sound: 'path_to_audio',
    event: 'mouseenter'
  });

  $('#button').easyAudio({
    sound: ['path_to_audio', 'path_to_audio'],
    event: 'mouseleave'
  });
```
